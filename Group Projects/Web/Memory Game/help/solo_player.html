Ajuda

Tutorial do modo: 1 Jogador

No modo 1 jogador, pode ser usado uma quantidade específica de cartas, dependendo da dificuldade que se queira jogar, desde que cada uma dessas cartas tenha uma e uma só outra carta igual para que se faça coorrespondência.

O jogador escolhe duas cartas, se forem iguais, ganha o par e essas 2 cartas desaparecem, se não, eles viram-se novamente para baixo. O jogo termina quando já não houver cartas para virar.
